# C# Console Quiz Game

A simple console-based quiz game built using C#.

## Features
- Multiple choice questions
- User input handling
- Score tracking system
- Console interface

## Technologies Used
- C#
- .NET Console Application

## How to Run

1. Open the project in Visual Studio
2. Run the program
3. Answer the quiz questions
4. Your final score will be displayed
